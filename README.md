# leofeng-ca.github.io

Leo Feng's personal academic website: [https://leofeng-ca.github.io](https://leofeng-ca.github.io)

Built with [Jekyll](https://jekyllrb.com/) on the [al-folio](https://github.com/alshedivat/al-folio) theme, trimmed down to About / CV / Publications / Teaching.

## Local development

```bash
bundle install
bundle exec jekyll serve --lsi
```

See [CLAUDE.md](CLAUDE.md) for where content lives (CV data, publications, about page) and how the site is structured.
